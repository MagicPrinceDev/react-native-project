### 二、题外话
我觉得React Native的发展是一种趋势，而且他最大的有点就是能实现跨平台共用代码，如果某个组件库不能同时兼容Android和iOS,我觉得就失去了React Native的优势，**所以本库中仅收录可以同时兼容Android和iOS平台的项目**。

**欢迎各位推荐好的 React Native 项目给我们**。
